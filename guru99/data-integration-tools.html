15 Best Data Integration Tools in 2020

Details
Last Updated: 20 March 2020
Data integration is the process of combining data from many different sources. It is used for analysis, business intelligence, reporting. Here is the list of top data integration tools with key features and download links.The list contains both open source(free) and commercial(paid) software.

1) ArcESB

Application and data integration doesn't have to be difficult, or expensive. ArcESB is a powerful, yet easy-to-use integration platform that helps users connect applications and data. It's intuitive, low-code flow designer takes you from installation to secure application and partner integration in under 30 minutes. Plug in, configure and go!

  • Completely FREE Core Automation framework
  • Visual, simple, fast, and secure application and data integration
  • Open architecture, enterprise-ready, without enterprise complexity
  • Certified secure managed file transfer and EDI integration
  • Runs anywhere – On-premise or in the Cloud
  • 100's of no-code connectors for ERP, CRM, marketing, finance, analytics, data warehouses

2) Improvado

Improvado is a "data pipeline" -- pulling data from your marketing platforms (Facebook, Google Analytics, Ad Servers, CRMs and email platforms, etc.) and "piping" it into any data warehouse or visualization tool you choose. Brands and agencies love using Improvado because it saves them hours of manual reporting time and millions in wasted marketing spend.

Features:

  • Integrates with 180+ marketing platforms
  • Aggregates all your marketing data into one place, in real time.
  • Integrations run deep so you can see data at a granular level -- by keywords, ad creative and more
  • Integrates with any existing business data
  • Ability to create custom metrics and map data across platforms
  • Complete support with a customer service rep included
  • No developers needed, it is plug and play

3) CloverDX

CloverDX is a suite of data integration technologies made for those who demand full, fine control over what they do, who need to solve complex problems in intensive environments, and who prefer to buy best-of-breed tools instead of developing their own.

Features:

  • Automate & orchestrate transformations and processes
  • Host in cloud or on-premise, scale across cores or cluster nodes
  • Code where needed
  • Collaborate between devs & less expensive teams
  • Co-exist nicely with existing complex IT environment
  • Build extensible frameworks to save money and share with colleagues
  • Enjoy enterprise grade personal support from CloverDX

4) Linx

Linx is a low code platform to easily build and manage integration processes for nearly any business system, including databases, APIs, files, third-party libraries, enterprise systems or custom apps.

  • Build, manage and deploy automated custom applications, code -free
  • Visually connect back-end data in databases, XML, HTML, Web services, JSON, or develop your own back-end as a service
  • Quickly build and reuse connections to bridge public and private environments
  • Orchestrate and transform data across systems with workflow and business logic
  • Create, publish, and manage APIs throughout their lifecycle
  • Separate run-time environment to deploy automated integrations and custom applications

5) Panoply

Panoply is a smart data warehouse that automates all three key aspects of the data analytics stack: data collection & transformation (ETL), database storage management, and query performance optimization

  • Over 100 pre-built automated data source integrations
  • Auto-scales by automatically allocating storage and compute requirements in real-time
  • Machine learning to automates joins and builds schemas
  • Automated query performance optimization with automated query materialization
  • Codeless management UI empowers non-technical users
  • Built on AWS architecture and SOC 2 certified and practices HIPAA guidelines
  • Connects to any BI visualization tool, e.g. Chartio, Looker, Tableau, PowerBI

6) Xplenty

Xplenty is a cloud-based ETL solution providing simple visualized data pipelines for automated data flows across a wide range of sources and destinations. The company's powerful on-platform transformation tools allow its customers to clean, normalize and transform their data while also adhering to compliance best practices.

Features

  • Centralize and prepare data for BI
  • Transfer and transform data between internal databases or data warehouses
  • Send additional third-party data to Heroku Postgres (and then to Salesforce via Heroku Connect) or directly to Salesforce.
  • Rest API connector to pull in data from any Rest API.

7) Integromat

Integromat is a powerful data integration tool with an easy to use interface. The integration editor is simple to use and robust enough to create complex workflows.

Features

  • HTTP,SOAP,JSON modules lets you connect with any data source
  • Create complex data integration workflows
  • Excel style functions to manipulate data.
  • Robust Error Handling
  • Aggregators and Iterators

DataDeck is a leading Data Integration tool to quickly connect, analyze, and share data with your team.

Features

  • 100+ readymade connectors including Slack, Facebook, Google Analytics and more
  • Ready to use templates

  • Set Goals across teams to monitor results
  • Access via Mobile
